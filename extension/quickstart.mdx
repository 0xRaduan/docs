Summate Chrome Extension allows you to summarize any website in seconds, be it youtube, blog post or any article online.
It is available to all subscribed users.

## 1. Download Chrome Extension

Please head to the [Chrome Web Store](https://chromewebstore.google.com/detail/summate-summarize-any-web/gckcmnmnijlejpodgccnblkpnlcflpmg) and install the extension.

## 3. Go to any website and refresh the page.

Sometimes you will see a message that you are logged in, but you still might need to refresh the page to get Summate working.

## 4. Click on Summarize Button or use keyboard shortcut

Once you refresh the page and click on Summarize button, you will see the summary of the page. You can do that on any website, and that should save you a lot of time.
