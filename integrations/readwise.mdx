Summate supports integration with [Readwise Reader](https://readwise.io/read)

#### Step 1: Generate Readwise API token

Go to [readwise.io/access_token](https://readwise.io/access_token) and generate your API token.

<Frame>
  <div className="flex flex-col gap-10 items-center">
    <img
      src="/images/integrations/readwise_1.png"
      alt="Generate Readwise API token"
    />
    <p style={{ fontSize: "0.9em", fontStyle: "italic", textAlign: "center" }}>
      Generate Readwise API token
    </p>
  </div>
</Frame>

#### Step 2: Save it on summate.io

After you generated your token, please go to [integrations settings](https://summate.io/settings/integrations) and save your API token.

<Frame>
  <div className="flex flex-col gap-10 items-center">
    <img
      src="/images/integrations/readwise_2.png"
      alt="Save Readwise API token"
    />
    <p style={{ fontSize: "0.9em", fontStyle: "italic", textAlign: "center" }}>
      Save Readwise API token
    </p>
  </div>
</Frame>

#### Step 3: Start saving your favorite emails

Once you’ve added Readwise token on the integrations page, you will be able to see a new action **“Save to Readwise”**.
